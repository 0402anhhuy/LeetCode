Các bài tập được phân loại theo **chủ đề** để dễ theo dõi và ôn tập lại, ví dụ:  

- 🔗 **Linked List**  
- 🔢 **Counting / Math**  
- 📊 **Array**  
- 🧩 **Dynamic Programming**  
- ⚙️ **HashMap / Set**  
- 🌳 **Tree / Graph**  
- ...

- Luyện tập tư duy thuật toán.  
- Cải thiện kỹ năng giải quyết vấn đề.  
- Chuẩn bị cho phỏng vấn kỹ thuật.
